# [Linux 프로세스 관리 명령어 정리] <br><br>
---
## 1. top - 실시간 프로세스 모니터링 <br>
> 현재 시스템에서 실행중인 **프로세스 상태를 실시간으로 확인**하는 명령어
> 특징 : 실시간 모니터링 -> 프로세스 상태를 계속 확인 가능

### 사용법
```bash

top
```
* 실행 후 명령어
   * `p` -> CPU 사용량 순 정렬
   * `M` -> 메모리 사용량 순 정렬
   * `q` -> 종료
---

## 2. ps - 현재 프로세스 상태 확인
> 현재 시스템에서 **실행 중인 프로세스 목록**을 보여주는 명령어
>
> CPU / 메모리 사용량, 프로세스 ID, 사용자 등 확인 가능

* 주요 옵션

| 옵션 | 설명 |
|-----|-----|
| ps aux | 모든 사용자와 모든 프로세스 출력 |
| ps -ef | 표 형태로 전체 프로세스 출력 |
| ps -u <사용자> | 특정 사용자의 프로세스 출력 |

### 사용법
```bash

ps aux
```

* ` ps ` - 한시점에서만 프로세스 상태 확인
* ` top ` - 실시간 상태 확인 및 모니터링
---
## 3. jobs - 쉘에서 실행 중인 작업 확인
> 현재 **쉘에서 실행되거나 중단된 작업** 확인
>
> 특징 : 다른 터미널에서 실행 중인 프로세스는 보이지 않

* 상태 표시

| 상태 | 의미 |
|-----|-----|
| Running | 백그라운드에서 실행 중 |
| Stopped | 중단된 상태 |

### 사용법
```bash

jobs
```
---

## 4. kill - 프로세스 종료
> 특정 PID를 가진 **프로세스를 종료**
>
> 정상 종료 혹은 강제 종료 가능
>
> 특징 : PID 확인 후 종료 -> `ps`나 `jobs`를 함께 사용

* 주요 옵션

| 옵션 | 설명 |
|-----|-----|
| kill <PID> | 정상 종료 시도 |
| kill -9 <PID> | 강제 종료 |

### 사용법
```bash

kill 2345     # PID 2345 프로세스 종료
kill -9 2345  # 강제 종료
```
